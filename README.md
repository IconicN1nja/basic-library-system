# 📚 Basic Library System (Java)

![Java](https://img.shields.io/badge/Java-OOP-orange)
![Status](https://img.shields.io/badge/Status-Active-success)
![Project Type](https://img.shields.io/badge/Project-Console_App-blue)

A simple **console-based library management system** built using Java.  
The program simulates core library operations such as managing books, issuing books, and returning them through a
menu-driven command-line interface.

---

## 📌 Project Overview

This project demonstrates the implementation of a **basic library management system** using Java fundamentals and
object-oriented design.

The system allows users to:

- View available books
- Issue a book
- Return a book
- Manage library records through a simple CLI interface

The goal of this project is to practice **Java programming concepts, data handling, and logical program structure**.

---

## 🛠 Tech Stack

- **Language:** Java
- **Concepts Used:**
    - Object-Oriented Programming (OOP)
    - Classes and Objects
    - Arrays / Collections
    - Conditional Statements
    - Loops
    - User Input Handling (`Scanner`)

## ▶️ How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/IconicN1nja/basic-library-system.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd basic-library-system
```

### 3️⃣ Compile the Program

```bash
javac *.java
```

### 4️⃣ Run the Program

```bash
java Main
```
